# curaos §5 - Agent Decision Weights (per ADR-0099 §10)

| Criterion | Weight |
|---|---|
| AI-agent friendliness | **5.0** |
| Developer experience (hot-reload, fast feedback) | **4.8** |
| Mainstream hiring pool | **3.6** |
| Performance + RAM footprint | implicit filter |
| Security posture | implicit filter |

When proposing any stack pick, evaluate against these weights. Do not propose without rationale.

See [[curaos-stack-priorities]] for full weight system + comfort-zone language list.
