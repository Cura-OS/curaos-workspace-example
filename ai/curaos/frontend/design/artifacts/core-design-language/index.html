Focus & states

  • Focus ring is a 3px primary-subtle halo + 1px primary edge - visible on any surface, never clipped (no overflow:hidden on focusable parents).
  • Never color-only. Status pairs an icon + label with its hue; table status uses a dot + text, not row tint alone.
  • Hit targets ≥ 44px on touch surfaces (clinician-app / patient-app); ≥ 32px dense desktop controls.

Dark-mode mapping

  • Primary alias maps 600 → 400 (brighter for low-light contrast); subtle maps 50 → #0a2030.
  • Borders become translucent light over solid dark - structure without noise.
  • Backgrounds use #0d1112 / foreground #eceff0 - never pure black/white.

Tenant override & RTL

  • Swap the --primary-* ramp at :root; semantic aliases + every component inherit. Neutrals + status stay fixed.
  • Layout uses logical properties (margin-inline, inset-inline) so RTL mirrors without overrides.
